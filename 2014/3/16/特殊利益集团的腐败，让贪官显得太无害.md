# 特殊利益集团的腐败，让贪官显得太无害

**法团主义的特殊利益集团的腐败，让贪官显得太无害**；

**法团主义的特殊利益集团是利益集团的子集，（特殊利益集团∈利益集团）**

**“反动的统治阶级不甘心退出历史舞台”是指法团主义的特殊利益集团；**

**利益集团可能自食其力，特殊利益集团依赖纳税人的成本生存；**

**特殊利益集团一定存在不合理；因此法团主义（特殊利益集团一定合理）天然地被证伪**；

远至通钢国企工人暴力事件，[到石燕氏纵火杀人事件中暴露的](../../../2012/8/31/“向弱者倾斜”是最伪善的美德.md)，垄断国企在臃肿中膨胀的人事组织，再到金融垄断集团对凯恩斯主义的坚持、对土地财政高房价的鼎力、光大乌龙指事件暴露的国有机构“作弊式市场”，以至于他们为了避免烂帐暴露，不惜滥用国家威权的种种“严打民间金融”所以维护的所谓“国家金融秩序”；无一不是后验着特殊利益集团之“法团主义”的“特权最大化”的本能。公众可以自已后验一下，所谓“反腐败，恨贪官”的狂热者，是否也多属于这类特权集团者？[与这些集体腐败相比，单个贪官算个鸟](../../../2012/8/31/让民主滚开！特权工人阶级不答应！.md)？

公有制总是口口声声地谴责于“个人的腐败，个人的奢侈”，直到体制外小民百姓的所谓奢侈，代替体制内的贪官个人，承受赤贫民粹的仇恨为止！——>[此即所谓“仇富”！](../../../2009/2/28/与既得利益者合理妥协，就是争取和平.md)但是体制内的贪官，就真的算多大的问题吗？非常有趣的逻辑，但却极少引起公有制文化中人警惕之处，在于公有制总是将“万恶之因”归于“个人”；而将“万善之由”归于“集体”；而丝毫不理会，在逻辑上必然成立的“逆结论”：既然个人是如此微小的，那么个人之“腐败，堕落”的影响也必定渺小；[集团之贪婪必定凶悍！集体之腐败必定远盖于贪官个人](../../../2010/2/26/“反政府”是荒谬的.md)。

按马克思主义阶级斗争式的反唇相讥，可以将体制外的“商人，知识分子，屯积（投机）者，农民，体制外工人……”等等也视为“同样是利益集团”，[似乎这样就变成了“利益集团的此派彼派，各自自利](../../../2009/7/30/十几亿体制外老百姓的利益由谁呼吁.md)”。辩证法万能！肯定可以让他们中的粉丝，相信特殊利益集团的自利也符合斯密《国富论》的精神。如果不考虑法团主义的基础是“国家干预市场，形成法团主义的垄断基础”的话，那么“利益集团总是需要代言人”，显然是正确的表述[；毕竟体制外的工商业者如民企，也的确是利益有共通处的“利益集团”，](../../../2010/9/25/国富论：“既得利益”是对社会的贡献.md)也需要“代言人”。

**ps:空灵转达的马恩毛狗型的话，他们对公务员加薪的合法性辩护就是：“[公务员要国家加薪，关体制外的纳税人什么事](../../../2014/1/25/什么是法团主义？公务员“加薪”中法团主义的自利.md)”**；

借此也正可以说明[，“特殊利益集团”并不是关乎道德的帽子](http://blog.sina.com.cn/s/blog_5563a64d0102egou.html)，而是对现实中利益分配的习惯规则中的描述。**特殊利益集团是利益集团的子集；数学上可以表述为（特殊利益集团∈利益集团）**。**特殊利益集团的特殊之处，就在于他们需要“国家损耗纳税人支付的成本的扶持”，才能平衡他们存在所需要的成本**。换言之，市场经济之中不存在特殊利益集团；[只要国家权力不干预市场，特殊利益集团也将自然消失](../../../2009/6/30/行政监管越多越腐败，从三鹿事件到上海塌楼.md)，——>当然，套用马克思主义阶级斗争中，此时颇为适用的话语，——>他们（特殊利益集团）是不甘心自动退出历史舞台滴。

特殊利益集团是损害纳税人的成本才得以存在；[因此特殊利益集团的本身，就是纳税人承受的税负痛苦中的一部](../../../2013/11/16/国企不能够卖，国企却无数次IPO，股市成了国企隐性税收工具.md)分，而无论是否显性地出现在税务清单上。特殊利益集团可以通过完全的税收和财政的支付，将其成本以政府和部门预算的方式，暴露在纳税人的审查之下，这是监管政府的最起码的和件。但是特殊利益集团也同样可以以国企、特权、行政垄断机关等形式，从显性税收中消失，但仍然作为隐性税收存在。由于公有制困境导致的低效（如上述暴露的特权最大化），[同一社会功能的隐性税收，将比其显性化后的税收要沉重得多](../../../2013/11/29/“什么是税负？”，明朝的隐性税负，明朝的解放军的奖金.md)！

**特殊利益集团并不一定是不合理的，如警察和军队**，[毕竟在任何国家之中都有适当存在的意义](../../../2013/1/4/中国古典的资本主义思想；经济衍生政治的关键环节；.md)。**但是特殊利益集团之中肯定存在不合理的，这就是法团主义本身不合理之处**。如果法团主义是合理的，将意味着“特权最大化是必然合法的，纳税人必须承认其任意鱼肉纳税人的合理性”，——>这种“必须承认”在法学上简单地表述为“奴役”！文学上有更灿烂的颂歌，被特殊利益集团称为“爱国主义”，[甚至可以找到美国总统的话当权威](../../../2009/7/28/不要问国家对你做了什么，要问你为国家做了什么.md)：“不要问国家（特殊利益集团）对你们干了什么，问你们为国家（特殊利益集团）贡献了什么”。
