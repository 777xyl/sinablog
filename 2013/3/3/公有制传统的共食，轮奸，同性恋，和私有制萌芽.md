# 公有制传统的共食，轮奸，同性恋，和私有制萌芽

标签： `共食` `公有制` `斯巴达` `轮奸` `同性恋` 

目录： `文明社会进化史`

**公有制传统的共食，轮奸，同性恋，分家和私有制的萌芽**；

拉西第梦人成年后就要负责自已团队的共食，大致是每年一次。共食是希腊军事社会的传统，具有军事性质，目前在于拉近全队感情。从荷马史诗透露的史前希腊的生活方式，已经可以看到希腊人（从北方南下的多利亚人）具有游牧军事部落的共食传统，每年的共食是部落认同的重要仪式。毛主席在文革大饥荒时，曾经强力推行“公共食堂”，具体决策冲动的原因，可能永远沉没在历史细节之中，[但是马克思主义的公有制即进步的理论](../../../2009/4/28/用阶段性社会发展史理解人类社会是误读社会进化.md)，掩盖了最基本的历史常识：共食（公共食堂）是[原始军事共产主义的特征性习俗](../../../2011/7/21/“原始共产主义”就是原始奴隶制.md)，自原始人开始即如是。

共食并不是希腊人所特有的军事传统，也是任何一个仍然具有部落军事传统的民族和文化所特有。可以这样说，共食本身代表着公有制的文化，分食本身就代表着私有制的萌芽。共食可以让团队对“共同食物供应短缺”的关注；分食则可以提高食品利用的效率，从而养活更多的人口，支持更高的经济水平。[共食的传统甚至可以追溯到猴群和狼群的进食次序](../../../2011/1/30/狼的斗牙和狗的斗嘴.md)，正是人类公有制社会“等级分界”的原始形式。当同一层次的狼和猴共同进食时，意味着其本身对低层次（后进食）者的优越，首领优先进食的本身，[也就是今天社会主义的特供](../../../2013/1/22/提高社会福利反而促进大革命到来.md)。

共食的传统广泛见诸于从蒙古到阿拉伯，再到北欧维京民族，也同样可见于中国古老的祭典，和在当时是非常珍贵的祭肉分配。祭肉之珍贵，不仅仅是因为在古代的“肉食”是非常难得的，（一直到清末民初时，中国人一年也只吃几次肉），更因为它意味着公有制集体的认同。节日和盛大庆典，或者婚姻丧葬，一般伴随着祭典仪式，在仪式后的大吃大喝，表面上的食品浪费，背后就是“我们是一伙的”重大认同。铁木真的母亲之所以一度离开孛儿只斤氏，就因为在过年祭典时，居然没分给他们家祭肉，也就是要轰他们走的意思了。

共食现象是人类历史上的公有制共同存在的事实，至于共食为什么成为共有制传统的特点，如果不是当作不解之迷，终归可以找到自已满意的后验理论。今天中国人爱大吃大渴，实际上就是共食的公有制传统。为什么没事找事，朋友之间就要吃一顿，绝不会是因为饥饿之故。今天一些经历过饥荒的，或者被粮食危机论灌残了大脑的学者（[如铅笔社的王志安先生](../../../2013/1/18/要废除行政垄断，但不要监管市场价格.md)），会对中国人共食聚餐中的“食品浪费”大惊小怪，其实在大饥荒时共食，无论是被彭德怀批评的公共食堂，还是普通朋友的聚餐，在当时条件下，都是是严重的食品浪费。

共食现象之中的食品浪费，本身就是为公有制的“凝聚力”所支付的代价。“共用”以增强凝聚力，在文明程度比较低的军队或部落之中，会扩展到“性共享”即集体强奸。[在古希腊则体现为集体的同性恋](../../../2010/8/5/西方古代公有制文化缺乏家庭元素.md)。马克思主义在共党宣言中，特别提到了“共食，共产，共妻”，可视为对古老传统的妥协。集体主义的性共享，经历过中日战争的历史的中国人，都比较熟悉与之相关的日本军队的有关细节，但常常忽略了中国军队也有类似的“性共享”，包括延安！分食代表着私有制萌芽，那么分妻（家族组成）则是私有制原子合作单位的确立；



了解共食现象在公有制传统，特别是在非职业性军队中，提供让指挥官垂涎的团队凝聚力的作用，就可以理解，为什么共食在拉西第梦人之中的重要性。职业性军队中的共食和共奸现象会弱化，但即使在美军之越南战争中看，也没有完全消失。在拉西第梦团队之中，不参加共食，几乎等同于自动退出斯巴达的统治阶层，而为团队提供共食一天！也成为每一个拉西第梦人及其家庭（转移到下属农奴，奴隶和属民身上）的沉重负担。责任者要按标准供应全队的饮食，[如果数量质量达不到国家标准，](../../../2012/5/5/公害知识分子的恶法之国家标准.md)拉西第梦人的公民身份就失去了。
