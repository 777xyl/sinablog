# “人性本私”是社会进化论入口,“人性本sth”是所有哲学的入口；

标签： `逻辑能力` `自然` `社会进化论` `哲学` `不信神` 

目录： `科学和世界观和进化论`

时间： `2012-12-08 21:01:10`

**“人性本sth”是所有哲学的逻辑入口；**

**“人性本私”是社会进化论的入口；**

**“人性本善，本恶”是社会进化论特定条件下的逻辑结论；**

**社会形态都是“如何抑制生物强盗本能”的不同实践**；

只要追根溯源，就会发现 “人性本sth”，几乎是所有哲学入口的断言。[象马克思主义的哲学根源，就是基督教及其迷信者奉行的“人性本恶论”。](../../../2010/12/27/路德新教是与马克思主义完全相反.md)因为人性本恶，因此私有财产就是万恶之源，因此就要共产主义。至于所谓“极大优越性”，只不过是为此崇高理想制造的奇幻文学。基督教对于“人性本恶”开出的处方，则是要信仰上帝；如果不信上帝，所谓不信神，则本恶的人性就会成为万恶之源。类似马克思主义，基督教就泡制了死后上天堂的神学。[基督教大致会同意温和的马克思主义派别，两者殊难区分](../../../2010/12/20/“开明专制”不可能长期稳定.md)，但一般反对极权主义。

基督教要求贵族和领袖信上帝，代表上帝向弱者倾斜，[但是否履行为上帝服务的贵族责任，由基督教会裁决](../../../2011/1/24/什么是法治？中世纪道德法庭公信力何来？.md)，以此作为承认贵族和领袖特权等级的条件，本身带有利诱、政治和宗教利益的合作，也隐含着“如果不就范，就判为异端，发动人民群众闹革命”的民粹威胁。基督教本身也的确以此方式，铲除了绝大部分异端，今天称之为宗教战争。但没有更多的根据，认为法西斯主义，极权主义，以及如文革式的民粹运动，本身就是基督教的政治主张。[尽管几者之间，从文化和政治渊源上，显然有着千丝万缕的关系](../../../2011/9/16/为什么基督教和马克思主义都攻击科学进化论？.md)。

各国传统文化中各式“人性本sth”哲学，[如孔儒的“人性本善”](../../../2009/6/26/精通马列和国学儒教才能扔掉这个主子吗？.md)，相当于基督教中的“仁慈，感化”的依据，总体上与马克思主义和基督教的观念，大同小异；并无基督教传教人士所称的，“西性本恶是文明，东方人性本善是愚昧”的根据。由于“人性本私”（即自利，自助，国富论）在语法类似“人性本sth”的哲学入口，也就让很多哲学家和迷信者，把社会进化论作为另一种哲学，而争论于“人性本什么”，或者用哲学争论的方式，在社会学和进化论中，辩证来辩证去，[或者干脆在歪曲进化论后](../../../2009/2/18/进化论的科学性；回应马恩基督教的质难.md)，攻击进化论和社会进化论本身。

“人性本私”的确是社会进化论的入口，但却不仅仅是断言，[而是生物进化论的逻辑结论](../../../2009/11/9/生物学，进化论，基督教和马克思主义.md)。汉语句法上的神似，但与哲学入口的绝对真理意义的“人性本sth”不同的是，“人性本私”是有外延限定的客观描述，至少有两个条件限制：1.针对个体；2.在进化论范围内，前两个条件就将得出第三个逻辑结论，“人性本私”服务于个体繁衍的私之利益。科学与哲学的不同之处，就在于科学尽管也有结论（哲学中有时称之为帽子），但命题和结论是有成立约束条件的。同理所有“人性本sth”，[如果附加相应的客观条件](../../../2009/11/18/绝对的真理之大尾巴狼定律.md)，也可见于进化论。

[孔儒中的“人性本善](../../../2008/9/4/“人之初性本善”之“老于世故”.md)”，只要附加“人之动物性的强盗本能得到约束”的条件就可以完全成立；[反之马克思主义和基督教的“人性本恶”论，](../../../2011/3/23/基督教不是人权的标准；美国不是民主的权威.md)附加“强盗本能的得益，不被有效约束和惩罚”的条件，也同样可以在进化论中成立。因为“交换创造价值，合作创造共同体，即共同利益的复合个体”，那么只要有办法在自然中限定“人性本恶”的强盗本能，“人性本善，人性本恶”似乎水火不容的对立断言，就可以在进化论的科学逻辑体系中，统一到“人性本私”的社会进化论之中，成为不同客观条件下的逻辑结论。

[民粹运动即“公有制民主”，](../../../2011/3/11/被民粹运动阻断的资本主义进程.md)默认逻辑前提是“人民和集体的本性是善的”“统治者个人的本性是恶的”；因此人民群众监督于统治者，当然就是万能的解决方案。这就是怪胎民主派的精神信仰。但其逻辑前提都是不成立的，“集体（复合个体）”的本性仍然是强盗本能，具体行为如特殊利益集团；统治者也远非个人而是一个集体。极权主义（在中国即商鞅的法家暴政）的观点是，“民众本性是恶的，因此要用君主和官僚逐级管理之”，但是却不能解决“统治者及其集团的强盗本能，如何监督？”的悖论！[基督教求助于上帝去了](../../../2010/12/20/基督教和马克思主义的社会行为如出一辙.md)。
