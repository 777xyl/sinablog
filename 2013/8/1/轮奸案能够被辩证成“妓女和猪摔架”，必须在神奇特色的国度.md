# 轮奸案能够被辩证成“妓女和猪摔架”，必须在神奇特色的国度

标签： `替罪羊` `司法腐败` `受害者` `李天一` `轮奸` `文化` 

目录： `传统文化国民劣根性`

**互联网栽赃批斗的流氓手段有效，李天一集团的流氓辩护也可能有效；**

**言论不自由，互联网受监管，左棍流氓就是能够“人至贱则无敌”；**

**中央集权司法不独立，流氓辩护就总比依法辩护更有效；**

**轮奸案能够被辩证成“妓女和猪摔架”，必须在神奇特色的国度**；



从李蒙记者开始，李天一辩护集团，铺天盖地的宣传，就一直给人以颠倒错乱之感。如果对互联网上以乌有之乡为代表的左棍，以及冒充精神病和女人的哲学家们，对他们的批斗大会手法有所接触的话，就会发现，李天一辩护集团，[用的就是批斗大会道德攻讦、人身攻击的抹黑手法](../../../2013/4/29/左棍大三阳.md)。如果你不反击，他们就说你是（心虚）默认；如果你反击，他们就说你是炒作，如果公众有你的同情者，[他们就说是网络暴力，让他们遭受到不公正的待遇](../../../2013/7/31/李家集团或故意激怒公众，为司法腐败创造条件.md)，然后就是封网啦，封博啦，封帐号啦，封法庭啦，非公开审判啦……，他们反而象“受害人”鸟！



互联网上被这些冒充精神病的左棍踢场，[配合“不明真相的管理层”宏观调控](../../../2013/5/13/毛左再创革命新底线，连管理员也分不清裁赃诬告.md)，因此被关掉的“右派”博客，可以说是不计其数。这些博主损失的是心血，这些左棍还能损失什么呢？可见互联网上，冒充精神病人颠倒黑白，引来管理层宏观调控“讲政治”，常常是有效的。由于言论自由得不到保障，因此而对政治当局导致的合法性损失，也就在事实上变成了，“只要毛狗民粹愿意冒充精神病，就能让任何统治者，成为被各门各派革命的对象”。[互联网上一直循环的口水游戏，](../../../2013/5/15/小管应适当偏袒博主，小管存在误判，小官存在误管.md)当局改革的命运，是否就这样被操纵在毛狗民粹的手中？

当李天一的辩护集团，把互联网流氓倒打一耙的伎俩，发挥得淋漓尽致时，[他们如果不是在法制内已经走逃无路](../../../2013/7/30/李天一辩护集团的炒作思路和误判.md)，就必定是对这套左棍谙熟的阶级斗争信赖有加。试问有多少人想到，李天一辩护集团故意挑战公众底线，故意刺激公众，有可能是让李天一轮奸无罪的王牌呢？从此案初次曝光，李蒙记者因为此嫌名叫李天一，就急不可待地跳出来指责公众，李天一辩护集团铺天盖地的全是“不准提李天一名字”，再到清华法学教授易延友先生声称“为回护李天一，政府必要时可封闭互联网”，[你能说这套流氓手段，在中国完全无效](../../../2013/7/31/李天一的悲情炒作，俺国特色既非法治，也非人治，是讲政治.md)吗？

此种流氓手段亦见于韩国的互联网，[大概因为韩国也有着“国家监管互联网](../../../2012/4/13/韩国的实名制“严打谣言”出尽洋相；.md)”。冒充精神病人反咬一口，能够成为我国左棍稻草人的杀手锏，归根到底也因为，我国官方保留了监管互联网的绝对权力。试问当新浪小管不再是各打五十大板，甚至于试图往笔者头上栽赃“黑社会”的铁杆的乌有之乡毛左，忽然之间神奇地拥有了删除不利于彼的贴子，冒充的精神病和没有冒充的铁杆毛狗，彼此配合得天衣无缝地要枪毙这个博客时，无论是我们又如何去鉴定，这只是几个毛狗地下党的革命，还是上溯到党国的政策呢？[此即所谓“耦合](../../../2013/7/31/讲政治的司法耦合，让李天一等同志得以轮奸执政合法性.md)”。

由于言论自由、司法透明、独立的司法不能讲政治，[所以这些观念在“西方那一套”里已经成了天经地义的习惯法](../../../2013/7/29/辛普森案，李庄案和李天一案的人治，法治和法制.md)，因此西方互联网上（骂街虽然也有）就没有冒充精神病，以封博封口为目的的革命水军；李天一辩护集团，抹黑受害人以求脱罪这一套，在西方恐怕只能起到适得其反的作用。以美国那一套而论，不要说未成年人轮奸这类严重犯罪，完全在成年人法庭审讯，[根本没有什么“保护未成年人犯罪特权”之说](../../../2013/7/11/李天一及律师有权“无耻”.md)；甚至于比尔盖茨年轻时开车超速这类小过错，也只是是法庭公告上不直提其名罢了。从来没有公众不允许知晓之说。

李天一辩护集团的道德炒作的脱罪策略，既是互联网上流氓手段，在司法辩护中的实践；也是我们这个传统的美德社会，从民众到统治者都没有“默认权益归于个体”这一最基本的民主意识的实证。脱离了“默认权益归属”的默认，[在互联网上就不能区分](../../../2013/5/15/避免关闭评论的一些好主意.md)“冒充精神病人攻击他人在先而招来的谴责”和“左棍对人不对事的人身攻击”的区别。在李天一轮奸这类司法案件上，当然就不能区分，轮奸者李天一和被轮奸的杨女士，到底谁是受害者了。轮奸案能够辩证成妓女和猪在道德泥潭里摔架，[也只有在传统美德的神奇国度才有可行性](../../../2010/7/21/不要搞道德“治法”.md)。
