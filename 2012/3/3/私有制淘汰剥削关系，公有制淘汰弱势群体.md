# 私有制淘汰剥削关系，公有制淘汰弱势群体

**社会进化论；私有制淘汰剥削关系，公有制淘汰弱势群体；**

**没有私权就没有公共利益；PublicRight与CharterRight的异同**；

得自上帝的君权Throne(马毛教称之为先进性)授予的Charter，总是自称代表了“公共的利益”。因此今天甚至一些口口声声要民主的人，也[立足于“民主是反腐败”的思路](../../../2012/3/1/民主的法学常识和私有制和自由.md)，要求Charter寡头贵族的廷议，首先要落实“不要空谈反腐，要落实公共财产(Charter)”，——>不是Charter贵族自已提出来，而是[被Charter践踏的贱民知识分子，所谓民主觉醒的公知分子自已！提出来的](../../../2008/10/16/极力维护不公平制度的是受害者自已.md)！

正如[强调PrivateRight的个体价值观](../../../2012/3/2/Charter特权的道德责任和民企PrivteRight自负成本.md)，被忠君的贱民卫道士们，攻击为“无政府主义”一样，“[反谷物法，AntiCharter”也被称为不要公共服务](../../../2012/2/14/韩寒引发的冷思考和方舟子卖力的热广告.md)。实际上个体社会是通过自由交换（合作）结合起来的。无论是对PrivateRight的强调，还是对未确实合作价值的外来人的拒绝，还是对约定契约的信诚守约，都是为了合作关系的形成和稳定；因此个体社会也是合作的社会。

因此个体社会与等级社会的差别，[不是“有没有合作”的关系，而是“如何形成合作”的关系](../../../2010/2/3/“斗争哲学”取代“务实合作”的传统文化.md)。同样在社会进化论上，个体社会的所谓淘汰弱者，也不是对个体的淘汰，而是对不合个体效益的合作关系的淘汰。不合个体效益的的合作关系，包括了从不合理的契约，一直到恶法，直到神仙皇帝和贵族集团，是[对所有这些不合公民个体效益的“合作方式”的淘汰](../../../2011/2/6/人权法治的汰恶留善“恶法能除”.md)。

所以个体社会可以总结为，为了个体至上的目的，而不惜淘汰任何合作关系，——>相比之下，被基督教和马克思主义这些[攻击社会进化论的信仰分子们](../../../2012/2/26/公有制的复古传统和革命的本质和进化论.md)所攻击的公有制，则是为了公有制的合作关系，不惜淘汰任何个体！淘汰方式从暴力的阶级斗争和内外的战争，一直到非暴力的和平时大饥荒，都是为了捍卫公有制和[代表公有制最高合法性的信仰（上帝），王权Throne，和贵族特权Charter](../../../2012/3/1/为什么信仰社会无法避免宗教战争？.md)。

[了解人权私有PrivateRight的至高无上](../../../2011/7/20/私有制与奴隶制的矛盾和unfair的含义.md)，和因此而形成的针对不合理的社会合作形式的淘汰；与信仰至高无上，并因此形成的为了信仰而不惜代价地，从贱民开始淘汰社会个体，更可以明白，公有制的信仰社会之[所谓“向弱者倾斜”到底是怎么样的高明厚黑](../../../2011/11/19/“人人平等”是私有制的专利.md)，也可以明白，当基督教和传统道德分子们，可以就什么活熊取胆杀狗吃肉之类的“动物保护”，反映出的是何等的愚民素质！

ps:只要中医熊胆疗效不被否定，禁止活熊取胆，将导致野生熊被盗猎。不否定中医却反对活熊取胆，伪君子的愚民嘴脸，让人刮目相看。

个体社会是针对合作关系的最优化，当然就不是不[需要作为合作形式一部分的“公共服务”。](../../../2011/10/19/公有制的税收，是绝对的权力.md)CharterRight的形式也存在于民主社会的公共领域Public，——>仅仅是形式！因为PublicRight的授权和范围，取决于所涉自治区域的[PrivateRight的授权和权力的让渡](../../../2012/1/2/阿罗不可能定理：公共服务有边际，政府不是越大越好.md)。因此其权力来源是自Private而不是自信仰Religon/God/Throne/Charter。这也是要把PublicRight与CharterRight加以区分的原因。

当[传统等级社会的卫道士们](../../../2010/10/13/中国传统文化对个人主义根本抵触.md)，攻击个体社会“自私自利，没有公共利益”时，他们实际上是偷换“公共，共和，Public”和“特权，贵族，寡头，Charter”之间的概念。实际上是只有要Charter特权，就不存在Public公共利益。因为Public是已经被私有化成为Charter，并由王权经上帝的同意，用武力加以维护，[而用道德加以约束](../../../2011/1/25/有中国特色的“罗伯特议事规则”和“对事不对人”.md)。Public只存在于能授彼之权的Private社会，No
Private No Public。
