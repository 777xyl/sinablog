# 横财收入为何在经济学上，一般是对本国产业的诅咒？

目录： `人权法学和经济学` 

时间： `2014-12-23 19:53:19` 

原文链接：[http://blog.sina.com.cn/s/blog_5563a64d0102vczr.html](http://blog.sina.com.cn/s/blog_5563a64d0102vczr.html)

**横财如石油收入及马歇尔计划等，不是收入国的国内产业需求；**

**石油美元、中国外汇储备和马歇尔计划，相当于对美国产业的政府补贴；**

**横财收入即便不消耗国内成本（如石油），仍然相当于补贴了国外产品的进口倾销；**

**美国秩序：因为美国产业领先好几代，因此马歇尔计划等可促使中国日本和欧洲，一定程度的产业升级**；

横财如石油收入，[可以将它看作是某种财富（或称资本](http://darthvad.blog.sohu.com/162201079.html)），不是欠发达国家的国内需求。作为交换媒介（马克思称之为“一般等价物”），它是提供此收入的国家横的国内产业的需求。[向欠发达国家提供横财外汇的，一般也是发达国家](../../../2009/2/14/外汇不是钱，是物资！“分国企，分外汇”难言吉凶.md)，只要它们才有能力开发新资源，并成为新资源的主要买主，如英国曾创造的煤和茶的需求，美国创造的石油和咖啡的需求。美国固然可以用买石油的方式支付美元，但事实上也可以用“信贷援助”的额度，向盟约国家提供类似的“横财收入”。典型如马歇尔计划，及向以色列提供军事采购的信贷额度。

**马歇尔等额外提供的信贷额度，并不是对受援国国内需求的补贴，而是对美国国内产业的补贴**。[因此马歇尔计划的真实效](../../../2014/10/28/麦克阿瑟的左倾改革，道奇路线的拨乱反正.md)果，是扩大了本来在战后处于过剩状态中的美国企业的，事实上是由美国政府财政补贴的销售；对盟国的军事信贷则构成了美国政府对自已军工行业的补贴。因为美国各方面的技术，确实领先任何盟国好几代，因此马歇尔计划也就能够起到了双赢的作用，让美国优势的产业得到更大优势的同时，也让盟国的产业升级，可以向美国提供更多的，也是更廉价的产品。[两者的互动就权成了二战后世界各国的增长要素](../../../2014/11/1/近代以来世界各国经济增长的美国因素和中国泡沫.md)。

**ps:可以对比中国今天五毛公知所称的“美元支援俄罗斯，南美，非洲”的“[新马歇尔＝资本项目输出”是什么东东](../../../2014/2/24/美国人投资全世界与中国的“资本项目输出”的根本不同.md)？请问它们是卖国？误国？还是爱国**？

因为横财收入不是后进国家的国内需求的满足物，相反这些后进国家为了得到横财，[总是要消耗国内的某种财富（对应着市场需求），](../../../2013/10/19/中国卖血过多的社会危机中的政府的“成本制高点”.md)以换得这些在统治家看来是真正财富的“横财”，——>只能采购发达国家的产品，也就是补贴发达国家的优势产业！而发达国家如美国，从来不反对后进国家转移走自已国内的夕阳产业！——>因此这些横财，对应是国内市场需求的损失。此所谓“社会主义国家，总是内需不足”！如果横财不能够进口更高的技术产业——>经济意义上必须是“产业”才有意义，[不是单项技术](../../../2009/2/17/外汇储备买不来先进技术.md)；——>则没有任何意义。

[马歇尔计划促使的战后世界的经济增长](../../../2014/11/21/从美国凯恩斯主义的宣传，观察美国特殊利益集团的左倾和自利.md)，以及日本和中国曾经的，也是自以为的“经济奇迹”
，原因正是因为后进国家可以用自已手上的美元，无论它是马歇尔计划赠予的，还是美国购买它们的资源产品换取的；将美国夕阳产业换回国内的原因。所谓夕阳产业，与技术先进与否无关，——>[夕阳不夕阳，一直与技术无关](../../../2009/2/17/有内需没垄断就会有先进技术.md)，这是持“生产力理论”的马克思主义者，极难理解的社会学常识，——>而与廉价劳动力，是否能创造在美国销售的成本优势有关。原因在于后进社会之所以是后进，因为它们除了人命廉价，比起美国再无其他优势。

所以美国等发达国家，绝对不可能廉价转让，他们没有落入竞争劣势中的技术，更遑论“先进行业”鸟。后进国家就算购入了美国的先进技术，因为它们国内缺乏同类需求，只能向美国出口，——>注意此时的市场条件：美国本土的同类企业，完全没有落入任何劣势，而处于所有方面的优势之中，[因此后进国家那怕自以为掌握了引进的先进技术](../../../2009/2/18/进口技术设备的用处就是腐败.md)，也不可能在美国本土市场取胜。而在后进国家本土，这些所谓先进产业，因为没有本土市场，在它们本土纯粹就是单纯接受投资的寄生虫。[所以习阿平称“外汇买技术”不可能解决任何问题](../../../2014/11/11/二战后的雅尔塔景气，局限性和经济增长边际.md)。

[因为横财如石油收入](../../../2014/12/20/资本定律：石油是魔鬼的排泄物，让社会主义国家更贫困.md)，不是本国产业满足本国产能的国内购买力；它只能用于进口别国的产品。结果就是，若将其补贴于国内某些产业的产能（一般也是夕阳产业，才会出现产能过剩，才需要补贴），结果不是造成本国落后产业的固化[（荷兰病的症状之一](../../../2014/12/18/俄罗斯重蹈西班牙帝国覆辙，俄国“荷兰病”的机理和传染性.md)），就是导致国内劣势产业在被补贴的进口产品冲击下衰退；**效果等同于本国政府，补贴外国产品，对本国行业的进口倾销**！该机理正是俄罗斯不能从苏联的灾难中，恢复自已农业的重要原因；也是阿拉伯国家的产业，一直保持在中世纪水平的原因。**但是横财外汇之害还不止于此**。
