# 利比亚战争中的吼猴主义和普世的进步“吼猴”；

**利比亚战争中的吼猴主义和普世主义的进步“吼猴”**；

**如果用“互联网的自由起哄，对进步主义推翻卡扎菲的普世革命，有什么用”的标准去衡量，答案将是“没什么用”；——>**[假如不将“反效果”
当作“没用”的话](../../../2013/10/4/“宣传是否有效”的衡量条件.md)。在卡扎菲控制下的利比亚之自由起哄，甚至可能加强了被北约和伊斯兰恐怖分子联合袭击下的卡扎菲的支持面；而在卡扎菲控制能力以外的，阿拉伯世界如卡塔尔半岛电视台的谣言，基本事实认定：1)它是马后炮；2)它是境外炮；因此必定与利比亚战争的爆发无关。卡塔尔半岛电视台为代表的“互联网宣传”（全算吧），目的是为美英法军事征服利比亚洗白，[就此点而言它是反效](http://darthvad.blog.sohu.com/323533624.html)果。

如果半岛电视台及弥漫全球的普世进步佬，它们通过互联网为利比亚战争的呐喊助威，这些自费自发的进步起哄是有效的，就不会有后来叙利亚决议被否决，也不会有今天我们对利比亚战争的全盘否定，[不会有美英法三巨头的下台](../../../2016/12/30/利比亚战争三巨头全部倒台，特朗普上台或促进反思.md)。半岛电视台的谣言，唯一的效果就是干扰了对利比亚战争的灾难性后果的止损，以至笔者预言的“潘多拉之盒”几乎无一遗漏地兑现，——>直到今天，灾难仍然远未终止！普世进步佬之所以断言“通过互联网的革命起哄，赶走了专制链条最薄弱一环的卡扎菲”，因为[它们与中华帝国同样的思维方式](../../../2017/1/15/政治正确对利比亚战争的错误解读.md)，希望那就是事实。

准确地说，进步的普世佬希望革命都是起哄来的；而中华帝国也的确恐惧，普世佬的起哄真能哄来革命！因为当中华帝国执政的革命先辈，仍然是普世佬的时侯，[它们也自以为自已的井冈山传奇](../../../2015/11/7/进步主义对宣传的错觉，旧制度和大革命的现实.md)，是习用了马克思主义的武林秘笈后“起哄”来的。以至于今天夺权钢穴后，还按武林秘笈的教导，专门保留一下“起哄部”，——>当然，时代不同了[，当皇帝以后就不能再象以前做流氓那样起哄](../../../2013/10/7/合法性守恒定理和宣传无效，及法理的概念.md)，调子变子嗓音却大了；就跟普世后辈们对吼吧！——>有点象汉光武帝刘秀上皇帝后，醒悟到“当豪强时可以窝藏强盗，当皇帝后可是要奖励‘强项令’才行噢”！

从三角博弈的地图可知，钢穴领导是进步右派，普世佬是进步左派，它们在政治学上的归类都是进步主义，因此他们都相信正义乃至革命是起哄“吼”来的，不是因为社会科学的“人性定理：经济决定政治，政治决定动乱（军事武力）”的逻辑延伸，所以按照生物学进化论的类比，[它们都可以归纳为“吼猴主义](../../../2013/9/30/“吼猴主义－细节理性主义”的宣传本能.md)”。钢穴领导对传承马克思主义的吼猴秘籍是如此信仰，以此于完全不愿面对“亡蒋者，蒋也”的历史事实；若非蒋介石本身也是普世的进步佬，井冈山亚种的吼猴，能够吼灭蒋家王朝？若非美英法牛刀杀鸡，[普世吼猴能够吼灭卡扎菲政](../../../2013/1/4/不要把《大革命和旧制度》读到狗肚子里去.md)权？

从[三角博弈的“政治视觉”已知](../../../2017/1/1/任何进步政权，都无法识别任何“左的进步威胁”；.md)，进步左派的普世佬之吼革命，乃是不得已而为之。普世吼狗乃是政治瞎子，瞎子嗓门大，“吼”反而变成唯一革命手段。但是已经夺权钢穴成正果的领导阶级，虽然进化前也是吼猴，但是当上专制统治者后还继续当吼猴，还吼着“俺有自信”，真那么有信仰，“有自信”，还需要吼吗？！原因恐怕是第1)未及时“向右转，转得还不够”，所以[不明白“亡蒋者蒋也”的历史教](../../../2013/10/3/有中国特色的左右派，都有着“宣传万能”的共识.md)训，不知道“外因通过内因起作用；政策若能遵从经济科学规律，民富国安，进步普世的吼，能吼来革命？”，第2)是领导的进步基因传承。

因为利比亚战争是外源输入，因此与卡扎菲内政好坏无关了，其实当时卡扎菲政府的财政状况，在阿拉伯世界中可能属于最良好的水平。如果不是美英法三国恰好是进步主义掌权，卡扎菲政府无论如何放松吼猴战线，利比亚政权都能稳如泰山。反之，卡[扎菲集团要自问](../../../2011/10/28/美英法是不道德的，卡扎菲是不冤枉的.md)，为什么在阿拉伯世界内外，都如此孤立无援？对于新中华帝国而言，更值得关注why冷战结束后，最进步的苏联作为外部敌人消失，次进步的新中国也“向右拐”后，[美英法欧洲政治，反而出现“向左拐”进步](../../../2016/12/1/“通往奴役之路”与经济周期的形成机理；.md)了，甚至要普世鸟？中华钢穴跟风“吼着向左转”，就能够蒙混过关吗？
