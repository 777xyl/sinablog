# 伊索寓言解读牛二定律的威力：公有制为什么不可能“公正”？

目录： `社会阶层分析利益博羿`

**伊索寓言：公有制为什么不可能存在“公正”**？

**牛二定律的威力，取决于“私有的后花园被公有化”**；没有读明白已经反复解释的公有制政治的博弈关系的人，会以为笔者是在跟一些左棍ID呕气，甚至可能以为，笔者是给管理员（很可能是不
无好意地）删几条郁闷。[实际上笔者是非常珍惜，此一系例实例所构成的后验](../../../2014/2/2/没有人权后花园的权力绝对，就必定代以专制权力的绝对.md)，恰恰就是笔者先验的政治模型的实证。因此笔者感到值得据此后验事例，解释明白在
公有制社会背景中，我们每个人在争取尽可能大的言论自由的空间时，如何尊重他人的对等权益的的方法的同时，[保护自已的人身安全](../../../2014/2/2/为什么民粹总是反科学？个人主义者的原则与沟通条令不可质疑.md)。



笔者利用在本博圈中翻滚的左棍，[也是为了展示（公有制＋牛二）的搭配威力](http://darthvad.blog.sohu.com/300733734.html)，解释本博是按照什么标准，认定和处置“歧议和异见”，以及因为左棍折腾，为了阻断牛二定律的作用过程，才不得不断然处置，因此才有了认定和处置“歧议和异见”，此标准又必然地按照“法家＋肃反”原则实施。这就如同两军对仗的战场，不存在可以横冲直撞的“中立者”，更不可能对任何疑似敌人“疑罪从有，司法审判”才能开火的仁慈。任何不懂得此最基本的道理者，也是最容易被敌方利用者；[就算这东西是一头蠢猪，它也是该死的](../../../2009/9/23/战场上没有大声疾呼的中间派.md)！

本博左棍现象，只是公有制社会的常态，实体经济学后的后验。小管可能与左棍有关或才关的不当干涉，则进一步证明了笔者的先验预期：民粹牛二之所以有威力，是
因为公有制权力的不当干预的重要社会原理。即里根曾说的“政府不解决问题，政府本身就是最根本的问题”，也是[公有制牛二定律，消灭公有制民主的表现形式和具体过程](../../../2014/2/2/从牛二定律，理解民主进程的最根本阻力.md)。根本原因就在于，政府任何专政动作，无论出发点何如，[都同样必须遵循（法家＋肃反）原则](../../../2009/8/21/古今肃反的道德观之成分决定立场论.md)。民主可以千万样不如伟大领袖的英明，但至少可以做到每样公共政策，都精确符合私有原则下的共识。

笔者无从揣测小管在本文中删贴的出发点，[尽管相信小管没有针对性的恶意](../../../2013/5/13/毛左再创革命新底线，连管理员也分不清裁赃诬告.md)，至少左棍的贴子也经常给删了。笔者也相信，很大程度上不是小管
勤政，而是上头言论监管的压力，让新浪不得不保留“举报”“接受上访”的功能。小管也不得不象中国政府一样，不得不在民粹的压力下“有作为”，只好各打五
十大板。笔者想说明的是，尽管笔者理解，小管并没有针对性的恶意，但是它实际上给了[极端如乌有之乡等牛二](../../../2013/5/13/标本型的毛左愤青的文革宣言.md)以明确的信息：他的折腾是有响应的，只要有恒心，[他的确可以最终导致任何他所不喜欢的“言论”被封杀](../../../2013/5/13/我们和茅于轼都不是强者，张宏良司马南他们才是强者.md)。



这个例子真是太说明[公有制民主的最根本的弊端](../../../2013/3/1/朋党政治永远是改革的，每一派相对于另一派，都是改革的；.md)了，举一个伊索寓言说明这个道理：

有一个瓦罐和一个铁罐，一起掉到河水的急流中。瓦罐看上去非常紧张，[铁罐就象管理员一样安慰它（不妨看作是权限更低的博主，如笔者等](../../../2013/5/15/小管应适当偏袒博主，小管存在误判，小官存在误管.md)）：别怕，我会关照你
的，——>没有理由怀疑铁罐的真诚。但是瓦罐这样回答：我担心的就是您啊！请您离我远一点，我才会有真正的安全。不妨将此语境，换成公有制帝国改革者，会不会也说“多数人主义的正义，离改革远点，改革才有真正的安全”，——>可能吗？毫无疑问的逻辑结果，如果不是改革者象皮诺切特一样坚持镇压“多数人正义”的关怀，[就必定只是同流合污于多数人主义](../../../2013/12/23/宣传不能制造愚民，政府本身就是愚民所缔造.md)。

乌有之乡之类的牛二民粹，无论多么邪恶，[他们如果出现在官方媒体，那只是众矢之的](../../../2013/11/13/宇宙真理强势崛起一年多，地狱法则的英明投机！.md)；与我们后花园无关。请注意“后花园”的言论权利与“公共媒体”的区别！左棍们能够翻腾，都是我们各自的后花园里，因此本来可以通过“对言论尺度的绝对专制”，轻易处理的小事。这就是私有制社会，强制个体权益不可侵犯的好
处。但是在公有制民主的言论环境中，指望管理员“公正，无私”地平衡牛二无事生非的冲突。我们每个人就会处于权力铁罐旁边，畏惧着被权力误伤的瓦罐。[因为“后花园”在传统文化中被公有化了](../../../2013/5/19/所谓“黑粉”，是独立思考的“人权粉”.md)。
