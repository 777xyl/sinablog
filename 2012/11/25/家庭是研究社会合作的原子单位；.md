# 家庭是研究社会合作的原子单位；

标签： `价值判断` `客观性` `挪用公款` `家庭` `社会合作` `财经` 

目录： `人权法学和经济学`

时间： `2012-11-25 22:23:06`

**家庭是研究社会合作的原子单位；**

**家庭是社会合作的微观的抽象模型；**

**赌博危害家庭的必要条件是“公有制”；**



只有在排除了诸如“某某某因为赌博so that verybad”和“中国的股市比外国的赌场都不如”，诸如此类不具参考意义的[，明显带着妖魔化色彩的气](../../../2011/10/22/原罪就是妖魔化，妖魔化就是扣帽子.md)话，才可以探讨同样具备“价格博弈因素”的炒股和赌博中，[可能真实存在于某些“弱势群体”的真实故事](../../../2009/5/5/万恶之源皆为善.md)，也因而发掘出被妖魔化的“炒股，赌博”的故事背后，到底是何种社会痼疾和经济风险。千万不要误会，笔者并不认为赌搏有利阁下家庭和身心健康。笔者本来从来不赌搏，连打麻将小赌也从未尝试。但是赌博只是对自已财产的害处，如果要对社会为害，还需要其他必要条件。

正如股票投资，无论涨跌多空盈亏，[只要是用自已私有产权或合法授权内去投资的](../../../2009/4/5/传说中的“市场的不理性”.md)，伤害的充其量是自已的利益，或者是委托人（授权者已经准备承担风险）的利益，并无可能危害社会其他人的利益，而无论其他人如何观点。因此彼此之间完全没有逻辑联系。赌博如果是用自已产权内的钱的话，中将（胜赌）相当于股市投资中“不预期股价上涨，却得到了上涨的收益”的意外之财带来的额外欣喜，——>但一般并不意味着对阁下的财政有积极意义，[能够因为中六合彩而改变自已贫穷生涯的，只是可以忽略的极极极之少数](../../../2012/11/1/股市比六合彩／纸黄金“更投机，更赌，更圈钱”的强盗逻辑.md)。

[如果是用自已的钱去赌，输了也只是自已的财产内的损失](../../../2012/1/7/“选择命运盒子的技术”和“打破命运盒子的科学”.md)，也同样不可能损害别人的利益。即以一般因为赌博而产生纠纷的家庭来说，如果夫妻双方是各自有私房钱的，其中一方[拿自已的私房钱去赌输了，也不会影响另一方的利益](../../../2010/3/30/中国人好赌的原因.md)。如果有家庭是因为赌博产生了纠纷矛盾甚至家庭悲剧的，可以断定，他们不是采用了“各有私房钱”的家庭理财模式，或者是赌钱一方窃取了家庭公款去赌。**家庭的赌博有害，可以作为“赌博为害”社会的原子版**。家庭中赌博一方拿公款去赌而伤害了家庭，社会上就会有官员挪用公款去赌而伤害了社会。

[家庭是社会合作的原子单位](../../../2012/3/11/进化论中的家庭和阶级，社会分工越细未必越发达.md)，为害于家庭或者社会的，到底是因为“有人赌博中输了自已的钱”，还是因为“有人挪用了公款，输了最亲密的或者是最信任的人钱”呢？假如家庭预算中采用的是私有制模型，（家庭对于社会是私的单位，对于夫妻和子女就是存公的单位），夫妻子女各持自已的私房钱，根据家庭共同预算和各自收入而分担份额，这是“西方那一套”的私有制模式。反之则是东方传统即西方过去那一套，“一切收入要归公”，家庭财政归属一位大男人主义的老公，或者行妻管严的老婆坐庄，[这就是公有制](../../../2012/4/11/通过家庭暴力理解历史中的奴隶制.md)。

[个体是价值判断的原子单位](../../../2010/1/21/人权是价值判断的原子单位.md)，家庭是社会合作的原子单位。个体判断是他人无法预测的，家庭合作基本模式是具自然效益的，也是目前法定的（一夫一妻）。因此不存在理性人模型的客观性，但[是家庭作为社会合作的微观的抽象模型是成立的](../../../2011/7/3/市场经济中，政府的天职是“不作为”.md)。家庭也是每一个体至少可以观察，一般都可以体验的微观现实。所以象“私有制”还是“公有制”好，从“宏观，哲学”的角度永远不会有结果，但是只要从家庭开始观察，私有制的合理性是不必争论的，马恩毛对私有制的妖魔化也是显见的；尽管不否定有少数家长专制下的幸福家庭。

所谓“公有制”还是“私有制”好，[各自完全可以在自已家庭里实践](../../../2011/7/4/家政市场是是研究人权交换的活实例.md)。只不过毛左之辈，多是口是心非之徒，不会真实披露他们的家庭财政模型。否则他们应该知道，他们所谓提倡的“新生活”，反对的就是家庭的公有制模式，称之为“封建（父权）家庭”如《雷雨》。但是到了社会上，这些僭占着特权的剥削阶级，就因为他们僭占着特权，[作为特殊利益集团，明目张胆地反对私有制](../../../2012/8/31/让民主滚开！特权工人阶级不答应！.md)！再以赌博而论，如果是在私有制的家庭，充其量就是把当月共同预算给赌没了，下个月家庭总统换人即可。但如果是公有制就完全不同了！
