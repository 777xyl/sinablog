# 社区自治让民众有共同的利益呼声

基层社区民主建设，渐进建设民主中国，已经成为[辩证的人权经济学](../../../2009/2/6/人权经济学.md)重要的子系列。这是走向民主宪中国最重要的实操认识，针对目前社会上，无论是左派，右派，还是糊涂虫稀泥派（学名中派），对此都有着错误的认识，因此，笔者不惜多费点文墨。本子系列有：

《[民主之路：不能一步到位全国直选](../../../2009/3/1/为什么不能一步到位全国直选？不能一步多党制？.md)》

《[人性本私；民主不是治腐败的速效灵丹](../../../2009/3/2/人性本私；民主不是治腐败的速效灵丹.md)》

以前还有一份很重要的观点文章，也说到这个问题《[户籍制度恰恰是民主人权在无自治权社会里的体现](../../../2008/7/5/户籍制度是社区自治权的要素替代.md)》。

这是中国民主之路，在“人民素质”，“革命传统”，“传统文化”，“没有民族工商业”……等等离奇古怪的反民主理由之外，一条真正的路！唯一的路！在许多装出左派、右派、中派模样，不知民主为何物的社会活动家的喧哗声中，告诉中国，真正的民主中国，是怎样构成的；是如何保证社会正常运转。告诉人们，这些中外台港的社会活动家，很多，只是用农民的名义作为工具，练习着喧哗鼓噪的，非但不是民主，实际上却是，伤害民主的左派民粹。告诉中国社会，为什么农民总是最穷的，城市贫民总是很穷的，为什么农村是没有希望的，为什么此前任何社会运动，农民和城市老百姓，都总是黄宗羲定律的受害者，一次比一次糟？

因为，农民没有自已的利益代言人。因为，城市老百姓，没有自已的利益代言人。他们的利益，只能既望于既得权力者，无私党朝野内外一致的天地良心。换言之，既望于这个群体里道德最差的那位的短边长短！不是只有暴民才嫌福利分配永远不够，暴官、暴王、暴党，那个不是希望自已的高福利能分配的！农民能不穷吗？老百姓能不苦吗？当人民的利益，只能由钦定官员的良心去保障时；今天称为“开明专制”，历史却为今人提供了一个参考：罗马帝国的保民官！保民官，曾经完美地保护了人民的利益，令社会不必再进步了吗？

为什么上述所有的对民主自治的理由都是荒唐可笑的？原因就在于，这些人不理解，“民主就是允许民众申明自已的利益”。他们大概以为，“民主就是人民做主”，————文盲！社区自治，就是允许拥有同样利益的民众，可以一起申明自已的利益！让这个民众利益之声，达到最大的音量，参与更高级的社会利益分配的博羿。这就是民主！除非能够证明“民众是没有自已的利益的”，“民众的利益天然就给某些奴隶主代表了”，否则，任何反对社区自治的“理由”，都是荒唐可笑！都是别有用心！

这条出路，就是让基层老百姓有自已的利益代言人，使他们有能力对抗黄宗羲定律的侵袭。这条出路，唯一的出路，就是从基层做起，在最小的单位里，让他们推举自已的利益代言人，让他们在小单位里，首先实现自治！而不用，“顾全大局”！从村镇做起，从街道做起；村镇可以选县；街道可以选区；县区可以选市、省……。

而事实上，“自治”并不一定局限于“居住”，这是对社区自治的“社区”的误解。宗族、同乡、信仰、生活方式……，所有趋同的利益群体，都可以结为自治性的社区。这样，我们就听到了西方认为是人权的基本表现之一的词句“迁移自由”。中国那些具有暴民运动情结的民粹活动家，把“迁移自由”作为反对《[户籍制度](../../../2008/7/5/户籍制度是社区自治权的要素替代.md)》的同时，也反对自治制度的借口，那是大错特错。

因此，中国的民主之路并不是暴民运动，也不是反日反美，不是单纯闹革命推翻无私党，暂时，也不是全国性的多党制。党禁，当然应该开放，但是，首先不是在全国级别；而应该在市县一级的级别。开放！新闻当然要开放，用新闻责任追索制，代替沙皇200年前创造新闻审查制，可以少养许多五毛五元，同时，保证新闻监督的真实性，时效性，服务于地方民主建设。而是，在基层首先让当地社会组织，能够为当地民众的根本利益意愿服务。

有人说，这不是回到民国初年的军阀吗？根本不是！区别就在于，这是在中央宪法、军队监督下的地方民主选举，权力在地方民众的手里，而不在一两个掌握着枪把子的无私军阀手里。地方民主选举之不是军阀，就如2000的德国，不是希特勒德国一样。都是德国，纳粹两个字，可不能随便放。

和平过渡到民主宪政，必须为无私党提供一条和平让权，确保全国平稳过渡；并且，让他们有通过选举，重新上台取得合法性的机会。以此，换取无私党在民主宪政上的妥协。无私党拒绝民主进程，将彻底失去合法性的机会，——人类历史上还没有出现过一个善终的专制皇朝！试问那一个皇朝开国之初，不是百炼成钢，为国为民的？笔者相信，历史总有一天会让无私党面临两种选择：要么接受人民的选择，即使暂时失利，也总有一天可以重新上台；要么，走波兰、沙皇、清朝、或者苏联的路，被强行推翻。所以，民主之路，也是为无私党设计了一条获得和平手段下的合法性的路。
