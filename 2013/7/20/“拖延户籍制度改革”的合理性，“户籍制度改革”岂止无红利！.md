# “拖延户籍制度改革”的合理性，“户籍制度改革”岂止无红利！

标签： `大革命` `户籍制度` `自治` `改革红利` `胡星斗` 

目录： `传统文化国民劣根性`

时间： `2013-07-20 21:18:07`

**李克强总理“拖延户籍制度改革”的合理性；**

**户籍制度改革是右派极权和左派民粹的共识；**

**通往奴役之路的户籍制度改革“红利”的欺骗性；**

**中国并不缺乏西方的迁移自由，中国缺少欧美的居民自治**；

在最关键的“[户籍制度千万不要作实质性改革”](../../../2010/3/6/为户籍制度正名，是民主启蒙的关键一环.md)的层面上，李克强先生的政策精神读来是模糊。无论是正反双方，都既未感到松了一口气，更谈不上感到满意。撇除句面表意，放开中小城市的户籍制度登记，无疑是正确，相当于中世纪时放开城镇居住准入；——>但中世纪没有放开城镇本地市民权（citizen）！事实上，中小城镇的户籍没有什么吸引力，今天在京上广嚷着[“反户籍制度”的主力军，99%就是来自二三线城市的户籍市民](../../../2010/4/29/声称代表农民的绝大部分不是农民.md)。换言之，这些城市的户籍登记一直是开放的，但一直被居民放弃；则所谓开放准入还能有什么效果？

包括[秦晖在内的民粹公知](../../../2012/6/27/民主社会中的“户籍制度”是地方国民主权.md)，一直把“反户籍制度”作为道德正义时，他们始终掩盖着一个显见的事实：如果不是因为城镇原住民，因为自治而获得“本地户籍利制（市民权）的红利”，则城镇户口甚至不会有足够的吸引力，吸引中世纪农村移民进入城市。一个被剥夺了户籍制度（自治）背后的居民福利的户籍，实际上是没有价值的，是被陆续放弃的。这就是反户籍制度的人士的逻辑错误。当他们把“废除户籍制度”作为城镇化的政策条件时，[实际上他们正在操持的却是“去城镇化”，](../../../2010/3/4/户籍制度是被错误攻击的民权制度.md)同时摧毁着中国城镇的工商业基础。

[户籍制度的所谓改革](../../../2013/6/19/“反户籍制度”打中公有制帝国的要害，也暴露了民粹的本性.md)，是政府内马尔萨斯主义的右派和党外的民粹主义的左派的共识，——>试问反对者是什么人？试问您曾经听见过，除了笔者以外，还有其他人坚决支持居民户籍制度的？但所谓户籍制度改革，却是在几乎没有舆论上的反对者，几乎在舆论上“获得党内外共识，获得政府内外的共识”时，却无法有什么进展，[您难道还未意识到其中跷蹊吗](../../../2009/3/9/如果没有户籍制度了，天堂就来临了吗？.md)？在这样的舆论条件下，在举国体制下“户籍制度改革政治正确下”，几十年“改革”毫无进展，则唯一的解释就是，户籍制度必有其合理之处，[反户籍制度必有其邪恶之处](../../../2012/3/4/为什么户籍制度背后的地方福利是私有财产PrivateRight？.md)。

[在一个没有居民自治制度的社会](../../../2009/3/3/社区自治让民众有共同的利益呼声.md)，却口口声声“反户籍制度”，几乎就是特权极右和民粹极左的试金石。事实上，特权极右是要剥夺居民私有权益，民粹极左是要吃大户。彼此殊途的利益目的，却能够得到“共识”，只要回想起毛主义“打土豪，分田地，国有化”的历史，您是否感到似曾相识？叶檀非常含糊的称“户籍制度阻挡了外地贫民的上升途径（她把进入别人世代生活的城市，称为上升）”“户籍居民享有住房，生活习惯，本地税后福利的不公平红利（她把私有财产视为‘不公平的特权’）”，[可见民粹的要害是反私有权益](../../../2009/9/6/户籍制度是中国全体平民的根本利益.md)。

从西方特别是基督教左派中，可以看到一些似乎“支持中国废除户籍制度”的“自由迁移”的理论。但是诸如胡星斗和秦晖等，[这些自称为“自由主义的公知](http://darthvad.blog.sohu.com/189376453.html)”在引证这些（主要来自欧洲）外来观点时，却隐瞒了“自由进入他人地方当农民工（没有户籍福利）”和“新来者没有获得户籍福利”的事实，并偷换了其中的概念。当他们号称“到一处即可登记社保ID”时，他们隐瞒了欧洲（和美国）实际上是到可随迁移登记纳税和缴付社保金的ID，——>是[掏钱付款而不是伸手要免费午餐的ID!](../../../2010/3/6/向移民倾斜，居民如何实现“安居乐业”呢.md)这种迁移自由中国政府早在三十年前就提供了。

[中国今天并不缺少西方的迁移自由](../../../2009/3/6/自由结社，社区自治和迁移自由.md)，中国缺乏的是中国几千年所没有的，西方几千年前就开始完备的居民自治！中国的所谓“左派”所视为中国之万恶的，就是西方之所以成为民主社会的居民自治制度，中国极权主义者，称这为“万恶的户籍制度”！——>这就是笔者最为担心之处！明明是中国已经获得了民主的进步，明明是剥夺居民合理权益的“反户籍制度”的大倒退！[明明是通往奴役之路的法西斯主义“反户籍制度改革](http://darthvad.blog.sohu.com/181097614.html)”，却披上了“民主进步”的鲜彩外衣，还指望得到“改革的红利”！那大概是要到地狱领取的魔鬼红利了。
