# 统治者不会拒绝民主，但通往奴役之路的可能性极大

标签： `统治者` `胡鞍钢` `通往奴役之路` `民粹` `公知` `文化` 

目录： `自由民主宪政通往奴役之路`

时间： `2013-07-20 21:14:02`

**统治者不会拒绝民主，但通往奴役之路的可能性极大**；

**左右派通往奴役之路的危险性，令专制显得合理，而无论其本意**；

追问“统治者”是不是真的要民主的，其实是全无意义。可以肯定地统治者是想解决一些问题，毕竟屁股下的炉火还在烧着。同时党内外的民粹革命同志，就是非常大的问题。[远在统治者是不是嘴巴上象毛主席那样“我民主，我革命](../../../2013/7/18/从温总理遭遇的误解，理解改革者的难处.md)”，更重要的是诸如市场经济、人权法治，这样的根本性政策的推动。因此笔者从来不担心，领导同志不会迎合民众“民主，进步”的意愿。连慈禧太后也推动改革，连北洋军阀也一口一句“我也是革命的，我支持民主”，中国近百年来不支持民主进步的领导人，[除了毛主席以外，恐怕没有第二个](../../../2009/7/3/看看毛主席是怎样发动文革反腐的.md)。

笔者担心的是，真心实意要民主的领导集团，无论是前任胡温还是现任习李，[在传统文化的误导下](../../../2013/5/25/传统文化中的暴民运动，农民起义，阶级斗争，亡天下.md)，再次忽略了西方民主成功的私有制要素，再次走进了五四运动式的“中央集权，富国强兵，公有制民主”的误区。同样也象五四运动，等到外树强敌，国内山穷水尽时，就左手抓汉奸，右手抓反革命，在自已人里找异端，重开一轮自相残杀。尽管我们无法理解领导集团的内心所想，但是从活跃的党内外公知的言论，可以看到这样一种沿着《通往奴役之路》“真心要改革”，[却改进了人间地狱的危险性](../../../2012/8/22/天堂，地狱，信仰，个人主义.md)，是非常大的。

中国积极改革的90%的可能结果是无功而返[，直到更严峻的形势下，出现更激进的改革](../../../2010/1/13/一字真经拖字诀，南辕北辙慢慢来.md)。还有9%的可能，就是更激进的改革，改进了地狱。真的改对并有效果的可能性，大概不超过1%！无论是从党外马尔萨斯主义的右派之张五常薛兆丰，还是从民粹左派的叶檀时寒冰胡星斗秦晖，以及民粹毛左的卢麒元张宏良司马南等公知身上，都显见其危险性。正是在这些人“通往奴役之路”的衬托下，党内原教旨面目的马屁精，如胡锡进和胡鞍钢等人，尽管其反民主反人权的言论让人作呕，[但相比于反复革命的折腾](../../../2011/11/4/独裁者未必真独裁，贪官未必真的是贪.md)，反而未必显得极恶之犹。

[由于上述党外左右派的误区](../../../2013/6/15/自然秩序不可能通过革命建构，“革命!多少复古以你为名”！.md)（他们都不是私有制人权这一端），及其建构主义万一被遂行的危险性，因此就模糊了“党内真正的特殊利益集团的战略上的保守”，还是避免大权旁落的“战术保守”者的区别。就笔者本人而言，尽管笔者所持的“人权私有的民主”被天朝视为敌对意识形态的根本异端，但是估计党内外的革命同志，因为笔者不革命的渐进立场，而将笔者视为“统治者的代言人，利益集团的代言人”，此类糊涂蛋也不在少数。实际上，笔者是改革战略上的激进派，因为目标是自然秩序；[但战术上是保守者，步步为营](../../../2012/2/1/横眉冷对伪君子，左狗总是闹革命.md)。

如果民间在野的观念，已经偏向于“私有财产神圣不可侵犯”的市场经济，[则薄熙来同志纵有天大的野心](../../../2013/7/18/从温总理遭遇的误解，理解改革者的难处.md)，又岂敢作重庆模式之冒天下之大不韪？既然连胡星斗秦晖等人也持“共同富裕”之薄熙来民粹，连薛兆丰叶檀也要对“非法集资”落井下石，更不要说卢麒元和张宏良之徒，早就声称要“镇压民营资本家”，那么又如何识别，如温家宝先生偶然表露的民粹，王歧山先生举起《旧制度和大革命》，习近平先生偶而流露的“左倾”，不是战术上的保守，而是战略上的倒退呢？用一句汉语旧俗而言[：（公知）已身不正，何以正他人](../../../2010/2/26/中国的民主只不过就是从自已做起，从现在做起.md)？

[或如胡锡进的理论自信](../../../2013/6/23/宇宙真理的汉语误会，自寻短见的理论自信.md)，清华胡鞍钢的人民社会优越论，无非是总体党领导下有无比优越性的同义反复。如果民众不是奉民粹为公知，这些特殊利益集团的马屁精，就根本不会创新出颠倒黑白的宇宙真理，给党国宣传拍马屁！则如胡鞍钢先生，也会保留其早几年的改革理论家的面纱，珍而惜之；不会瞅着新一任领导的脾胃，摇身一变成了“马列毛原教旨主义者”。[反之，如果民粹重复几句免费午餐的远大理想](../../../2009/7/27/实用主义的现代愚民制造业.md)，愚民大众就屁颠屁颠地认了公知！则胡鞍钢教授这么聪明的人，掺点民粹当公知，搞点人民社会的辩证法，向领导请赏去鸟。

**思考题：清华胡鞍钢先生，用无可争议的个人哲学，论证了“苏联朝鲜古巴纳粹这类人民社会，比欧美公民社会更先进”；犯了那些逻辑错误**？
